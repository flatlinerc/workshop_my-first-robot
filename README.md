# My First Robot

My First Robot is a workshop to give people an introduction into 'robotics'.  It consists of off the shelf parts, a 3d printed chassis, a software suite, teachers guide, and associated files.

This is designed to be expedient and made cheaply and quickly so that students can get an idea of how things can be done with 3d printers and the ready supply of hobbyist ready components.

BOM:

- ESP32 microcontroller
- VL53L0X ToF distance sensor
- DRV8833 dual motor driver breakout board
- 3d printed chassis
- M3 screws of varying sizes
- Dupont style jumper wires
